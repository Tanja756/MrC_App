# MrC_App

Локальное Flask-приложения для работы с заявками ТОиР. Данные синхронизируются с Яндекс.Диском, все действия сохраняются локально в SQLite и при наличии интернета выгружаются обратно на Я.Диск.

## Быстрый старт

```bash
# Linux
chmod +x run.sh && ./run.sh

# Windows
run.bat
```

Скрипт сам создаст виртуальное окружение, установит зависимости и запустит сервер на `http://127.0.0.1:5000`.

## Настройка

Перед первым запуском откройте в браузере **Настройки → Яндекс.Диск**:
- **Токен** — OAuth-токен Яндекс.Диска (получить на https://oauth.yandex.ru)
- **Пользователь** — ваше имя (для отметки в заявках)

После сохранения нажмите **Загрузить данные** — скачает текущие заявки, справочники, склад, ППР и график ФН.

## Как это работает

- **Загрузка** — вручную или раз в N минут скачивает JSON-файлы с Яндекс.Диска
- **Работа** — все операции (взять/закрыть/отклонить заявку) создают action-файлы локально
- **Выгрузка** — action-файлы отправляются на Яндекс.Диск, откуда их забирает сервер MrC_WebApp
